# Missing-Semester-Class1

本博客用于记录我对missing-semester的学习

## 常用指令

### date

获取当前时间

### echo

把后面的第一个参数输出给终端

```bash
echo hello
```

上述代码将输出`hello`

```bash
echo $PATH
```

将咨询环境变量，找出路径，给出了echo指令的位置

### which

找出执行指令的位置

```bash
which echo
```

输出：`/bin/echo`

### pwd

输出当前工作目录

### cd

进入指定工作目录，其中`.`表示进入当前这个目录，`..`表示回到上一个目录

### ls

输出当前工作环境下所有文件与文件夹（不包含隐藏文件）

如需输出隐藏文件需加后缀`-a`

若需要详细列出文件与文件夹信息，使用后缀`-l`，并且在后面加上文件名，获取指定文件信息

### man

获取对应指令的详细操作手册

### cat

用于输出文件内容

### sudo

放在指令前，赋予根用户权限

### chmod

给文件改变权限

```bash
chmod +x filename
```

使文件具有可执行权限

### touch

更新文件的修改时间为现在的时间，如若文件不存在，则创建一个新文件

## 一些小知识点

### >

用于把前面的内容存储到后面的文件中

```bash
echo hello > hello.txt
```

可以把`hello`写入文件

### <

用于把文件内容进行输出

```bash
cat < hello.txt
```

把文件的内容输出出来

### |

把前面的指令的输出塞到后面程序的输入里面

### 一些特殊的符号若要写入文件需要用单引号包住，双引号不行
